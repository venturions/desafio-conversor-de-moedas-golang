# Tarefas

Use este checklist para ajudar a organizar a sua entrega.

- [X] Criar um diretório para o projeto e, dentro dele, o arquivo `main.go`
- [X] No `main.go`, declarar o `package main` e a função `func main()`
- [X] Adicionar a estrutura de dados `map` ou importar arquivo JSON com as taxas de conversão
- [X] Implementar a leitura dos argumentos passados pela linha de comando
- [X] Converter o argumento de valor (string) para um número (float64)
- [X] Realizar o cálculo de conversão multiplicando o valor pela taxa encontrada
- [X] Imprimir o resultado final no console, formatado com duas casas decimais
- [X] Compilar o código, gerando um executável chamado `convert`
- [X] Testar a aplicação com exemplos válidos (ex: `./convert 10 USD`)
